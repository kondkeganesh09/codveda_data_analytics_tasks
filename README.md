CodeVeda Tasks - Level 1 & 2
This repository contains my data science projects for Level 1 (Basic) and Level 2 (Intermediate) tasks.

## Level 1 - Basic

1. Data Cleaning and Preprocessing
* Handling missing values, duplicates, and inconsistent formats
* Tools: Python, pandas

2. Exploratory Data Analysis (EDA)
* Summary statistics, correlation analysis, data visualization
* Tools: Python, pandas, matplotlib, seaborn

3. Basic Data Visualization
* Bar plots, line charts, scatter plots
* Tools: Python, matplotlib, seaborn

## Level 2 - Intermediate

1. Regression Analysis
* Linear regression, model evaluation using R-squared and MSE
* Tools: Python, scikit-learn, pandas

2. Time Series Analysis
* Trend and seasonality detection, moving average smoothing
* Tools: Python, pandas, matplotlib, statsmodels

3. K-Means Clustering
* Standardization, elbow method, cluster visualization
* Tools: Python, scikit-learn, matplotlib, seaborn
